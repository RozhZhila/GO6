## Key Actors

| Actor | Description |
|--------|--------------|
| **Student** | A verified student user can create a profile, post and apply for opportunities, and collaborate on projects. |
| **Organization** | Verified groups such as companies, NGOs, research centers, or student initiatives that post jobs, internships, or projects for students. |
| **Administrator** | Developers/moderators who verify accounts, manage reports, and maintain the platform's security and functionality. |
| **External Service** | Third-party systems that integrate with the platform, such as email authentication services, file storage systems, or APIs for verification & notifications. |
